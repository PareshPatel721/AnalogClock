# Analog Clock

A beautiful analog clock built with SwiftUI.

## Features
- Real-time clock updates
- Smooth animations
- Responsive design
- Works on iOS, macOS, and watchOS

## Requirements
- iOS 26.0+ 
- Xcode 26.0+
- Swift 5.5+
